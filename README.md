# DC-p0t

Vulnerability verification

Honeypot on Docker container

## env

Ubuntu 20.10 (Linux 5.8.0-33-generic)

VirtualBox 6.1.16

## How to use

### packages

```
$ sudo apt update
$ sudo apt install -y bpfcc-tools
$ sudo apt install -y docker.io
```

### start

```
$ ./start.sh
```

### stop

```
$ ./stop.sh
```
